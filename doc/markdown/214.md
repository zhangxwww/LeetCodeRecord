# Shortest Palindrome

Difficulty: Hard

---

题目要求在给定字符串前添加字符将其转换为回文串。既然目标得到最短回文串，那就是要求添加的字符尽量少，也就是添加前后字符串尽量匹配。

所以可以用到一些字符串匹配算法，例如KMP，将原始字符串作为模式串p，将翻转后的字符串作为目标串t。对p, t做字符串匹配。

显然二者肯定不匹配(不然输入字符串本身就是回文串了)，但是如果二者都是从左向右遍历，匹配失败左移模式串，那么可以根据匹配结果知道二者之间的偏差(即模式串需要右移多少字符，才能与目标串匹配)。

得到这一偏差后，只需要在输入串之前添加对应的字符即可。
